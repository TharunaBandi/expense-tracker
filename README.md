# Expense Tracker

A full-stack Expense Tracker application built using:
- React (Frontend)
- Spring Boot (Backend)
- Java, JavaScript, CSS, HTML
- Maven
- Git & GitHub

## Features
- User login
- Add expenses
- Filter by month and year
- Delete expenses
- Responsive UI

## How to run
1. Start backend (Spring Boot)
2. Start frontend (npm start)
3. Open http://localhost:3000
